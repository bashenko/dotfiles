# dotfiles

My personal terminal setup: Hyper + zsh + tmux + Starship.

## Install

```bash
git clone git@github.com:bashenko/dotfiles.git ~/dotfiles
cd ~/dotfiles
./install.sh
```

## Stack
- **Terminal**: Hyper
- **Shell**: zsh + Oh My Zsh
- **Prompt**: Starship
- **Multiplexer**: tmux + catppuccin
- **Font**: JetBrains Mono
